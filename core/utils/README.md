# Core / Utils

## Propósito

Utilidades compartidas y transversales para todo CJ_OS.

## Responsabilidades

- Helpers comunes: fechas, strings, validaciones, parsing.
- Funciones de logging centralizado.
- Utilidades de serialización y deserialización.
- Helpers para manejo de archivos y rutas.
- Funciones reutilizables que no pertenecen a ningún módulo específico.

## Regla

- Ninguna utilidad en este directorio debe depender de lógica de negocio de módulos.
- Debe ser posible probar cada utilidad de forma aislada.
