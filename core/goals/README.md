# Core / Goals

## Propósito

Definir y gestionar los objetivos de la empresa.

## Responsabilidades

- Almacenar metas de negocio, KPIs y planes de mejora.
- Alinear las decisiones del sistema con los objetivos estratégicos.
- Medir el impacto de las acciones automatizadas.
- Permitir ajustar prioridades según objetivos cambiantes.

## Pregunta que responde

> **¿Qué intenta lograr la empresa?**

## Ejemplo

- "Reducir faltantes de inventario en un 20%."
- "Mejorar el tiempo de respuesta a cotizaciones."

## Fuentes esperadas

- PostgreSQL para objetivos estructurados.
- `config/` para objetivos por ambiente.
