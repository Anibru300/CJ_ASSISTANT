# Notas temporales de KES/SIC

Esta carpeta contiene las notas en bruto de cada Knowledge Engineering Session.

## Reglas

1. **No son documentos oficiales.** Son borradores de trabajo.
2. **Un archivo por sesión:** `KES-<numero>-<dominio>-notas.md`.
3. **Después de la aprobación, su contenido válido se consolida en los documentos maestros.**
4. **Las notas aprobadas se archivan; las descartadas se eliminan tras 30 días.**

## Plantilla de archivo

Ver `docs/CJ_KNOWLEDGE_ACQUISITION_PROTOCOL.md`.
