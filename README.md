# CJ_OS (CJ Operating System)

> Nombre temporal de carpeta: `CJ_Assistant`.  
> El renombre oficial a `CJ_OS` ocurrirá en la versión estable v1.0.

CJ_OS es el sistema operativo inteligente de 3P. Orquesta personas, procesos, datos y sistemas externos mediante automatización, eventos e inteligencia artificial.

## Objetivo

Crear una plataforma centralizada que reciba información desde múltiples canales (WhatsApp, Outlook, etc.), la procese mediante agentes de IA especializados, consulte sistemas internos y externos, y genere acciones, alertas, reportes y respaldos de forma automática.

## Reglas del proyecto

- No avanzar al siguiente Milestone sin validación de Carlos.
- Cada cambio debe ser reversible y quedar documentado.
- Configuración 100% externa (`.env` + `config/`).
- Archivos fuera de la base de datos; datos persistentes fuera de Docker.
