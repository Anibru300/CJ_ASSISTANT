# Known Unknowns — CJ_OS

**Versión:** 0.1.0  
**Fecha:** 2026-06-25  
**Estado:** Registro vivo — se actualiza durante cada KES  
**Propósito:** Listar lo que sabemos que todavía no sabemos, para no olvidar preguntas críticas durante las sesiones.

---

## 1. Principio

> **No es vergonzoso no saber algo. Es peligroso no saber que no lo sabes.**

Este registro captura preguntas abiertas, vacíos de conocimiento y decisiones pendientes. Cada ítem debe tener un responsable y una vía de cierre.

## 4. Estados

| Estado | Significado |
|--------|-------------|
| **Pendiente** | Aún no se responde. |
| **En investigación** | Se está recolectando evidencia. |
| **Respondido** | Ya se tiene respuesta validada. |
| **Descartado** | No aplica o no es relevante. |

---

## 5. Reglas de uso

1. Toda pregunta sin respuesta durante una KES se registra aquí.
2. Ninguna KES se cierra si hay ítems de alta prioridad sin vía de cierre.
3. Los ítems respondidos se transfieren al documento maestro correspondiente.
4. Este registro se revisa al inicio de cada sesión.
